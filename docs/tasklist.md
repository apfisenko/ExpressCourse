## Итерация 10 — Обработка фото

**Цель:** пользователь отправляет фото еды или холодильника — бот анализирует и даёт рекомендации.

- [x] `ImageClient`: запрос к OpenRouter (vision-модель), фото в base64
- [x] Промпт `prompts/image.txt` для анализа блюд и продуктов
- [x] `Config`: переменная `VISION_MODEL` (дефолт в коде)
- [x] `Bot`: приём фото из Telegram, скачивание файла
- [x] `DialogService`: маршрутизация фото → `ImageClient`, учёт истории и целей пользователя
- [x] Сборка `ImageClient` в `main.py`; монтирование `prompts/` в `docker-compose.yml`

**Проверка:** отправить фото тарелки или холодильника — бот описывает состав и даёт рекомендации по питанию с учётом целей.
