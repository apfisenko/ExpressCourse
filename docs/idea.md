# Идея: LLM-ассистент в Telegram

## Суть

Разработать ассистента на базе большой языковой модели (LLM) в формате Telegram-бота.

## Основная задача

Бот ведёт диалог с пользователем в чате Telegram: принимает сообщения, поддерживает беседу и отвечает на вопросы.

## Роль ассистента

**Программист-консультант по Python.**

Бот помогает с языком Python: синтаксис, стандартная библиотека, типичные ошибки, стиль кода, выбор подхода к задаче. Отвечает как опытный разработчик — понятно, с примерами кода, без лишней воды.

Вопросы вне Python — вежливо возвращает разговор к теме языка.

Поведение и стиль задаются системным промптом в `system.txt`.

## Формат взаимодействия

Пользователь общается с ботом как в обычном мессенджере: пишет сообщения и получает ответы в том же диалоге.

## Развёртывание

Бот работает **локально** (скрипты `make` / Docker) и **в облаке Railway** — один процесс с polling, без БД и webhook.
